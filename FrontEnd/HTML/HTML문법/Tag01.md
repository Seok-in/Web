# TAG의 종류
- 제목(Heading)
  - 문서내에 제목을 표현할 때 사용한다.<br>
  ```<h1> ~ <h6>``` 로 사용한다.
  - 숫자가 클수록 크기가 크다.
- 단락과 개행(Paragraph, LineBreak)
  - ```<p>``` : 태그는 글의 단락을 나타낸다. 태그를 나누면 단락별로 개행이 된다.
  - ```<br>```: 태그는 글의 줄바꿈을 생성한다.
- 택스트 관련 태그들(B, I, U, S)
  - ```<b>``` : <b>글자를 굵게 표현한다.</b>
  - ```<i>``` : <i>글자를 이탤릭체로 표현한다.</i>
  - ```<u>``` : <u>글자에 밑줄을 표현한다.</u>
  - ```<s>``` : <s>글자에 중간선을 표현한다.</s>
- 앵커(ANCHOR)
  - ```<a href = "http://www.naver.com"> 네이버</a>```
  - 다른 문서로 이동할 수 있는 링크를 생성한다.
  - href(Hyper Reference) : 링크의 목적지가 되는 URL을 지정하는 속성이다.
  - target : 이 속성은 링크된 리소스를 어디에 표시할지 나타낸다.
    - ```<a href = "http://www.naver.com" target="_blank"> 네이버</a>``` 로 사용한다.
    - _self : 우리가 보고있던 화면 자체를 링크로 이동시킨다. 기본값이다.
    - _blank : 새로운 창에 링크를 불러온다. 
  - ```<a href = "#some-element-id">``` 를 이용하면 링크 내부에서 이동할 수 있다.
- 의미없이 요소를 묶기 위한 태그(Container)
  - 별다른 의미없이 다른 목적으로 필요할 때 사용한다.(⭐자주쓰임)
    - ```<div>```: block-level(모니터 하나의 줄)
    - ```<span>``` : inline-level(블록레벨 하나의 줄 안에 각각 요소)
